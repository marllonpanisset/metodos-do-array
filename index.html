Treinando Javascript - Metodos do Array

concat() - junta 2 arrays join() - listar valores do array em formato string push() - adiciona elementos no final do array pop() - remove o ultimo elemento do array shift() - remove o primeiro elemento do array unshift() - adiciona um novo elemento no inicio do array slice() - extrai uma parte do array e transforma em um novo array splice() - extrai uma parte do array e transforma em um novo array reverse() - inverter os dados do array
